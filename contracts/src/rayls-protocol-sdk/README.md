
# Rayls Protocol
Rayls Solidity Cross-Chain Messaging Protocol.

## Installation

# via pnpm
```$ pnpm add @rayls/contracts ```


# via npm


```$ npm install @rayls/contracts --save ```

# Directory Structure

@rayls/contracts

```
├── src # Solidity contracts
│   └── v0.1
└── abi # ABI json output
    └── v0.1
```
